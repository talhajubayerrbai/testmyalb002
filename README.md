# testmyalb002

FastAPI application — bootstrapped by UDAP.

## Project structure

```
├── main.py                    # Entry point (uvicorn.run)
├── app/
│   ├── main.py                # FastAPI app factory + router includes
│   ├── config.py              # App settings (reads from env)
│   ├── routers/
│   │   ├── health.py          # GET /health
│   │   └── api.py             # GET /api/info — add routes here
│   ├── models/                # SQLAlchemy / Beanie models
│   └── schemas/               # Pydantic schemas
├── public/                    # Static files (optional)
├── Dockerfile
├── .env.example
└── .github/workflows/         # CI/CD (generated by UDAP)
```

## Local development

```bash
cp .env.example .env        # fill in your values
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload      # visit http://localhost:8000
```

## API endpoints

| Method | Path        | Description          |
|--------|-------------|----------------------|
| GET    | /health     | Health check         |
| GET    | /api/info   | App info + DB status |
| GET    | /docs       | Swagger UI           |

## Deployment

Push to `main` — GitHub Actions will provision infrastructure and deploy automatically.
Check `.github/workflows/deploy.yml` for the pipeline definition.
